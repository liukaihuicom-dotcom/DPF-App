# 2025-05-25

## FeedbackToast 组件重构

- **去掉图标背景**：`AppIconFrame` → `AppIcon`，图标直接渲染无圆形背景色
- **新增 variant prop**：`'title'`（仅标题）| `'body'`（标题+正文），不传时根据 message 自动检测
- **紧凑边距**：paddingHorizontal 24→12px (`spacing.md`)，paddingVertical 16→8px (`spacing.sm`)，gap 16→8px (`spacing.sm`)
- **图标 tone 映射**：success→success, danger→danger, warning→warning, default→white
- **新增 subtle 边框**：0.5px (`lineWidth.hairline`) + `colors.border.subtle`，解决 dark 模式下 toast 与背景层次不够的问题
- **外层间距**：host paddingHorizontal 12px→16px (`spacing.lg`)，与 screenPaddingX 对齐，留出呼吸空间
- **图标容器缩减**：badge 56px→32px (`size.control.lg`→`size.icon.lg`)，匹配图标实际 canvas 尺寸避免多余空白
- TypeScript 编译通过，向后兼容，所有现有调用无需修改
- 修改文件：`src/components/feedback/FeedbackToast.tsx`
