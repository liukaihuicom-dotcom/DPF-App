# Production Delivery Docs

This directory initializes the production delivery package for the Dupoin FX/CFD Expo app.

The docs follow the three supplied skills in this order:

1. AI Product Production Delivery Skill v1.2: product model, roles, rules, state, API, risk, QA.
2. Design System Production Governance Skill v1.0.0: tokens, components, business components, patterns, quality.
3. Visual UI Build Skill v1.0.0: page composition, state coverage, visual QA, pattern extraction.

No demo page is created here. These assets define the production guardrails used before any new UI page is implemented.

Start with `01-product-modules.md` for the current product function map / product brain. Use `02-page-map.md` to trace those modules to runtime pages, then use `00-project-structure.md` for ownership boundaries and directory rules.
