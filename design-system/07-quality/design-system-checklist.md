# Design System Checklist

| Check | Required |
|---|---:|
| Product role and state are defined | yes |
| Token mapping exists | yes |
| Shared component is reused or extended | yes |
| Business component is extracted when repeated | yes |
| Loading, empty, error, disabled, success, failed states exist | yes |
| i18n keys exist in both locales | yes |
| A11y labels and states exist | yes |
| Risk and security notes exist for financial flows | yes |
| Version record updated | yes |
