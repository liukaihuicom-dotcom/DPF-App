# Icon Usage

| Context | Size | Color |
|---|---:|---|
| Bottom tab | 20 | Active brand, inactive text dim |
| Status entry | 15 | Brand |
| Inline metric | 16 | Text muted or state color |
| CTA leading icon | 20 | Inherit button text color |
| Empty/error state | 24 | Text dim or state color |

Do not use icons as the only indicator for financial up/down movement.
