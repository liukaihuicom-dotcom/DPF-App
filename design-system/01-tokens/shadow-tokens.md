# Shadow Tokens

| Token | Runtime Value | Use |
|---|---|---|
| shadow.panel | `src/theme/colors.ts` `shadows.panel` | Highlighted cards and elevated panels |
| shadow.toast | `src/theme/colors.ts` `shadows.toast` | Toast notifications |

## Rules

- Use shadows sparingly in financial UI.
- Prefer borders and surface contrast for hierarchy.
- Dialogs, sheets, and floating panels may use stronger elevation only after token definition.
