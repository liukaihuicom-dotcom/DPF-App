# Z-index Tokens

| Token | Use |
|---|---|
| z.base | Screens and cards |
| z.sticky | Sticky footer and tab bar |
| z.sheet | Bottom sheet |
| z.toast | Toast notifications |
| z.console | Product control panel |

## Rule

Layered UI should be hosted globally where possible, as with `GlobalBottomSheetHost` and `ToastProvider`.
