# Motion Tokens

| Token | Value | Use |
|---|---:|---|
| motion.fast | 120 ms | Press feedback and small state changes |
| motion.standard | 180 ms | Sheet and toast transitions |
| motion.slow | 240 ms | Page-level transitions |

## Rules

- Motion must clarify state and feedback.
- Do not animate market numbers in ways that reduce readability.
- Respect reduced motion if a platform preference is exposed.
