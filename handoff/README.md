# Handoff Package

This directory stores production handoff templates and page-specific delivery packages.

For a new page, create:

```txt
handoff/<module>/<page>/
  page-spec.md
  field-spec.md
  state-spec.md
  interaction-spec.md
  component-usage.md
  api-contract.md
  i18n-keys.md
  accessibility-spec.md
  performance-spec.md
  security-risk-spec.md
  acceptance-checklist.md
```

Use the files in `handoff/templates/` as the source templates.
