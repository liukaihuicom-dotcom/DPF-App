# React Native Mapping

| Design-system asset | Runtime entry |
|---|---|
| Token index | `design-system-engineering/01_tokens/tokens.json` |
| Runtime tokens | `src/theme/tokens.ts` |
| Runtime colors | `src/theme/colors.ts` |
| Component manifest | `design-system-engineering/02_components/component-manifest.json` |
| Business component manifest | `design-system-engineering/03_business_components/business-component-manifest.json` |
| Icon registry | `design-system-engineering/04_icons/icon-registry.json` and `src/icons/iconRegistry.ts` |
| Pattern registry | `design-system-engineering/05_patterns/pattern-registry.json` |

React Native UI must use registered components and theme helpers rather than page-local visual primitives.
