# Icon Changelog

## 1.0.0 - 2026-05-22

- Added L5 icon registry for the used Phosphor linear subset.
- Added semantic icon naming, usage rules, core list, schema, and QA rules.
- Separated brand assets from functional icons.
- Established app icon quality gate for Expo 1024x1024 launcher assets.

## 2.0.0 - 2026-05-22

- Upgraded runtime provider to Hugeicons Free Stroke Rounded for higher-quality financial icon assets.
- Added Pro-ready provider metadata for Hugeicons Pro Duotone Rounded procurement.
- Replaced generic chart/bank semantics with candlestick, wallet, transaction, security, and KYC-specific provider icons.
- Added finance semantic scoring and provider validation to icon QA.
