# Icon Core List

The v2 core set contains 37 semantic product icons rendered through Hugeicons. Runtime uses Hugeicons Free Stroke Rounded; the registry is ready for Hugeicons Pro Duotone Rounded procurement.

Core categories:

- Market: `marketTrend` uses candlestick imagery; `quoteList` and `globalMarket` support watchlists and global market entry.
- Trading: `taskChecklist` and `historyClock` support order execution and transaction history.
- Funds: `accountBank`, `transferSwitch`, and `qrCode` use wallet/exchange/QR imagery instead of generic bank-only imagery.
- Risk and KYC: `riskShield`, `secureLock`, `identityCard`, `statusVerified`, and `infoCircle` use security and verification imagery.
- System navigation: `navigateBack`, `navigateNext`, `expandDown`, `closeX`, `moreDots`, `searchGlass`, `settingsSliders`, `notificationBell`.
- Support and growth: `supportHeadset`, `communityChat`, `chatFeedback`, `educationCap`, `rewardGift`, `promoTicket`, `achievementTrophy`, `partnerNetwork`, `emailMessage`.
- Account: `userProfile`, `userAvatar`, `addUser`.

No unused functional icons are allowed in the registry. Low-finance imagery is restricted to support/operations surfaces.
