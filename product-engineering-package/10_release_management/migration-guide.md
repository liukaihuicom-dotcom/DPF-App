# Migration Guide

This package does not migrate runtime data or code.

Before implementation:

1. Confirm payment providers and payout rails.
2. Confirm KYC, AML, fee, limit, and FX policy.
3. Convert OpenAPI draft into backend-owned API contract.
4. Build provider callback and ledger reconciliation contracts.
5. Run Financial Copy, Design System, UI Build, and UX Gate workflows.
6. Add automated tests mapped from `09_traceability_and_tests/test-case.mapping.md`.
